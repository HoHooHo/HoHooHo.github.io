  sudo rm -fr /Library/Internet\ Plug-Ins/JavaAppletPlugin.plugin 
  sudo rm -fr /Library/PreferencesPanes/JavaControlPanel.prefpane

注意版本号

xxx.jdk 中的 xxx是你电脑安装的JDK版本
